# 在终端运行该程序的步骤如下：

===== 运行步骤 =====

第一步：进入项目目录
cd D:\A-Freedo\研究型任务\ArchitectSenior\my-banana-workflow

第二步：激活虚拟环境
venv\Scripts\activate

第三步：把你的图片放入 inputs\ 文件夹

第四步：运行
python agent.py --image inputs/view1.png inputs/view2.png inputs/view3.png inputs/view4.png

===== 可选参数 =====

指定PPT风格：
python agent.py --image inputs/view1.png inputs/view2.png inputs/view3.png inputs/view4.png --style architecture --topic "空间智能研究报告"

指定页数（在config.yaml里改 total_slides）

去 outputs\ 文件夹找结果
